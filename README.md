
# Student Grade Calculator

# Project Description
This project is a simple Python program that takes marks of multiple students, calculates their average, assigns grades, and gives basic comments.

# Features
- Student marks input
- Average calculation
- Grade assignment
- Result display
- Class statistics

# Technologies Used
- Python
